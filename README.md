Тестирование сотрудников РТРС
Веб-приложение для проведения тестирования сотрудников РТРС. Система позволяет автоматизировать процесс проверки знаний, организовать удобное хранение тестов и результатов, а также разграничить права доступа для обычных сотрудников и преподавателей.

Стек технологий
Backend: Python, Django

База данных: PostgreSQL

Frontend: HTML, CSS

Функционал
Для всех пользователей
Регистрация и авторизация

Просмотр доступных тестов

Прохождение тестов

Просмотр результатов тестирования

Для преподавателей
Создание и редактирование тестов

Просмотр результатов тестирования сотрудников

Установка и запуск
Клонировать репозиторий

Установить зависимости: pip install -r requirements.txt

Настроить PostgreSQL в settings.py

Выполнить миграции: python manage.py migrate

Запустить сервер: py manage.py runserver
